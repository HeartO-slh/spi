CPOL CPHA MODE
 0    0    0      上升沿采样，下降沿变数，初始时钟为低
 0    1    1      下降沿采样，上升沿变数，初始时钟为低
 1    0    2      下降沿采样，上升沿变数，初始时钟为高
 1    1    3      上升沿采样，下降沿变数，初始时钟为高、
 
tx_mode0 在片选有效，等待半个sclk之后,再翻转sclk,上升沿在数据中间
tx_mode1 在片选有效，翻转sclk,下降沿在数据中间
tx_mode2 在片选有效，等待半个sclk之后,再翻转sclk,下降沿在数据中间
tx_mode3 在片选有效，翻转sclk,上升沿在数据中间

mode1提前于mode0半个sclk,使得采样跳变沿变化
mode3提前于mode2半个sclk,使得采样跳变沿变化

mode1、3 V1.0同样CS半个周期后，sclk翻转，加入flag;
mode1、3 V1.1 CS有效后翻转sclk,去掉flag，减少信号，但所用资源多于V1.0
